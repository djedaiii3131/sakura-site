📋 Blocs HTML pour Skool

6 blocs prêts à coller dans l'éditeur HTML de chaque leçon Skool · Automatisation TikTok avec l'IA

Comment coller dans Skool

1

Dans ton Classroom Skool → ouvre une leçon → clique sur le bouton "+" dans l'éditeur

2

Choisis "Embed" ou l'icône "</>" (HTML block)

3

Clique "Copier le bloc HTML" ci-dessous, colle dans l'éditeur Skool

4

Clique Save — le contenu s'affiche directement dans ta leçon ✅
